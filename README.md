# Realtime Voice Changer

This is a starter kit for a real-time voice changer.  
It allows you to upload a target voice and later run a model to convert your microphone input into that voice.

## Folders

- /server : backend code (server)  
- /training : instructions for preparing your target voice  
- /voice_uploads : place your voice files here  
- /models : place trained model files here
